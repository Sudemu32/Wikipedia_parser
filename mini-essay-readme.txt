Репозиторий:
-monitoring.py            скрипт который отслеживает актуальные записи
-parse_and_save.py        скрипт создает базу данных 
-config_settings.py       параметры для работы скриптов 
-person_id.csv            бд со скриптами


Парсинг осуществлеятся по данным за 2025 год, возможно расширить до 2006 года, после меняется стандарт разметки страниц Википедии

Перед запуском:
	1)Открыть config_settings.py и указаать адрес почты для атрибута RECIPIENT 
	2)Через командную строку запустить скрипт monitoring.py
			###НЕКОТОРЫЕ ЗАПИСИ ОТСУТСТВУЮТ ПРИ ЗАПУСКЕ СКРИПТА НА ПОЧТУ УКАЗАННУЮ В config_settings.py БУДУТ ОТПРАВЛЕННы СООБЩЕНИЯ С НЕДОСТАЮЩИМИ ДАННЫМИ###
